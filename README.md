## 📡 Project 3 – Distance Detection System

### Description
This project uses an ultrasonic sensor (HC-SR04) to measure distance and control an LED.

- Object near → LED ON  
- Object far → LED OFF  

### Components
- Arduino Uno  
- Ultrasonic Sensor (HC-SR04)  
- LED  
- Resistor  
- Breadboard  

### Pin Configuration
- TRIG → Pin 12  
- ECHO → Pin 9  
- LED → Pin 2
